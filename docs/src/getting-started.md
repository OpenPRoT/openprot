# Getting Started

## Prerequisites

- Rust 1.70 or later
- Cargo

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd openprot
```

Build the project:

```bash
cargo xtask build
```

Run tests:

```bash
cargo xtask test
```
